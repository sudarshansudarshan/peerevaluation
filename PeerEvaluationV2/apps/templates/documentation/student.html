For Professors

Registration/Login

Professors do not need to register themselves. Instead, they will receive login credentials from the admin.

After receiving the login credentials, professors can log in as a professor/teacher for the application.

Functionalities

  • Dashboard: The professor dashboard is where a professor can look into active batches, enrolled students concerning batch and course, and look into TeacherAssistants- can also be enrolled through the button Enroll TA.
  • Hover over buttons to see a description of their functionalities. Use the navigation button at the top-left corner to explore other options.

  • Profile: Access your profile page to view account details and use the logout feature.

Exam Scheduling

This functionality is only available when a course has been allotted to the professor by the admin.

View when no course is allotted
If you see this please ask admin to allot you course

View when a course is allotted

After a course is allotted, professors can add exam ,like python here, details such as the date, duration, maximum marks, number of questions, and the number of evaluations required. Once complete, click the "Save" button.

Upload and Send

This functionality is only available after scheduling exams.

Uploading Scanned Answersheets

Here now we can see that the upload option is available for the professor, the answer sheet of all the students from the batch can be uploaded in bulk using the upload button.

View when you have scheduled the exam
If you don't see this please ask admin to check.

Available Batches Option:

Under the available batches now we can see some buttons, if you hover over the button you can see the function of the buttons.

Button Descriptions:

  1. Download all answer sheets that are uploaded as PDFs.
  2. Send papers for peer evaluation after all the upload is complete.
  3. Mark evaluations as complete.click when all the students have done evaluation and we have a good mean for answers.
  4. Delete all examination data.Please make copy of data you need before using this.
  5. View analytics and download results. Through this you can view all the analytics and graph and also download the results during the evaluation till the evaluation is completed.
